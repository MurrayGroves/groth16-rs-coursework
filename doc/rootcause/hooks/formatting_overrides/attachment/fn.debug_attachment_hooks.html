pub fn debug_attachment_hooks(f: impl FnMut(&dyn Display))
Expand description

Calls a function for each registered attachment formatting hook for debugging purposes.

This utility function allows you to inspect all currently registered attachment formatting hooks. Each hook provides information about the hook type, the attachment type it handles, and where it was registered.

ยงArguments

  • f - A function that will be called once for each registered hook with a displayable representation of the hook information

ยงWarning

This function will lock the internal hook registry for reading, so it can potentially cause deadlocks if register_attachment_hook is called while the function is executing.
