Struct Polynomial 

Source 
pub struct Polynomial<F>
where
    F: Field,
{ /* private fields */ }
Expand description

Represents a polynomial over a finite field. Supports scalar and polynomial addition, subtraction, multiplication, division. Can be generated via Lagrangian interpolation over a vector.
